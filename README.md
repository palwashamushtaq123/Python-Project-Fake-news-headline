# **Fake News Headline Generator (Python Project)**

**Project Overview**

This is a fun and creative Python project that generates random fake news headlines using the random module.
The goal of this project is to practice Python fundamentals such as:

- Loops
- Lists
- Random selection
- User input handling

**It simulates breaking news headlines in a humorous way by combining random subjects, actions, and locations.**

**How It Works**

The program randomly selects:

- A Subject (e.g., Atif Aslam, Russian Cat, CM Maryam Nawaz)
- An Action (e.g., Eats, Dances with, Declares war on)
- A Place/Thing (e.g., parliament, sabzi market, IPL match)

**Then it generates a funny Breaking News headline like:**

# **BREAKING NEWS: A Russian Cat Dances with at Pakistan gate.**

# **The program continues generating headlines until the user types "no".**

**Example Output**

- BREAKING NEWS: Atif Aslam Celebrates inside parliament
- Do you want another headline? (yes/no): yes
- BREAKING NEWS: A Group of Monkeys Orders at sabzi market

**Concepts Used**
- Python Lists
- Random Module (random.choice)
- While Loop
- String Formatting (f-strings)
- User Input Handling
